# Project Overview

## Project Name

**isla-suds** (v2025.7.3)

## Purpose

Isla Suds is a Shopify Hydrogen-based headless commerce storefront for an artisan soap business. It provides a modern, fast, and SEO-friendly e-commerce experience built on edge computing infrastructure.

## Key Features

- Server-side rendered (SSR) React application
- Shopify Storefront API integration for product catalog and cart management
- Customer Account API for user authentication and order management
- Deployed to Shopify Oxygen (Cloudflare Workers) for global edge delivery
- Theme switcher for customizable appearance
- Predictive search functionality
- Full cart and checkout integration with Shopify

## Target Platform

- Runtime: Shopify Oxygen (Cloudflare Workers)
- Node.js: >= 18.0.0
- Package Manager: pnpm

## External Services

- **Shopify Storefront API**: GraphQL API for product data, collections, cart operations
- **Shopify Customer Account API**: Customer authentication, profile, orders, addresses
- **Shopify Checkout**: Hosted checkout experience
